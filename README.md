# 🚀 Lifeline Frontend

This is the redesigned Flutter frontend for the Lifeline project with improved UI, optimized structure, and better performance.

---

## 📦 Prerequisites

Make sure you have the following installed:

* Flutter SDK (latest stable version)
* Dart
* Android Studio / VS Code
* Git

Check installation:

```bash
flutter --version
```

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/lifeline-project/lifeline_frontend.git
cd lifeline_frontend
```

---

### 2. Switch to the correct branch

```bash
git checkout frontend-redesign
```

---

### 3. Get dependencies

```bash
flutter pub get
```

---

### 4. Run the application

```bash
flutter run
```

---

## 📱 Running the App

* Connect a physical device OR start an emulator
* Then run:

```bash
flutter run
```
